Materiales

Base coat y Top coat: indispensables para adherencia y brillo.

Nivelador de Ph y Adherente: previene el levantamiento prematuro.

Lámpara LED/UV: para curar el gel (48W o más).

Lampara uv de pre-curado: Para fijar rapidamente los tips de soft gel sobre la uña, antes de exponer la mano completa a una lampara principal de curado.

Tips de Soft gel: Tip completo para pegarlo en las uñas.

Empujador y Alicate para cutícula: para empujar y cortar la cutícula.

Gama de colores semipermanentes: Para aplicarle color a tus uñas.

Pinceles de arte para uñas: Para perfeccionar y hacer diseños.

Aceite de cuticula: Hidrata la cuticula.
